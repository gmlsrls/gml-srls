☁️ Sincronizzazione Online

I dati vengono sincronizzati automaticamente su tutti i PC collegati.

⬜ Non configurato — inserisci le credenziali Firebase qui sotto.
📋 Come configurare (gratis, 5 minuti):
  1. Vai su console.firebase.google.com
  2. Crea un nuovo progetto (nome qualsiasi)
  3. Clicca Firestore Database → Crea database → Modalità test
  4. Vai in Impostazioni progetto → copia API Key e Project ID
  5. Incolla i valori qui sopra e clicca Salva e Connetti
  6. Installa l'app su tutti i PC e inserisci le stesse credenziali
